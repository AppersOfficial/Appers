Why is HashTag under maintenance?

There are several reasons that could have resulted in the app being stopped and going down for maintenance. The biggest reason could have been a result of bugs, such as bugs that are not possible to be resolved by users directly. So, in order for the app to continue to be used without any problems, it is in maintenance, as the developers are improving it for a possible next update.

How can I report a bug?

You can report a bug directly from the app, or if you prefer, go to the bug report page.
